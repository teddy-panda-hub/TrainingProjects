Container Tags:

Container tags - These tags which don't have their own content eg.,
  • table
  • body
  • head
  • div

Standalone Tags:

Standalone tags - do not require a closing tag. They automatically end after their attributes are defined
  • br
  • img
  • hr

Content Tags:

Content tags - which must be opened and closed. The content is entered in between the opening and closing tags eg.,
  • h1...h6
  • p
  • b
  • i

Parent Tags:

Parent tags contain one or more child tags, creats nested structure defining the document's hierarchy eg.,
  • table
  • tr
  • td

Attribute Tags:

Tags with attributes provide additional information or modify the behavior of an element eg.,
  • id
  • class
  • style
