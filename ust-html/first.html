These tags which don't have their own content eg., table, body, head, div Standalone tags in HTML, also known as self-closing tags, do not require a closing tag. They automatically end after their attributes are defined eg., br, img, hr Content tags are used to structure and display the content of a webpage. They encompass text, images, and other elements, defining the layout and hierarchy of the information which must be opened and closed, the content is entered in between the opening and closing tags eg., h1, td, p, b, i Parent tags contain one or more child tags, creating a nested structure that defines the document's hierarchy eg., table,tr,td Tags with attributes provide additional information or modify the behavior of an element eg., id, class, style
